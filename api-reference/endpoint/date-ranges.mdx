---
title: 'Check Available Date Ranges'
description: 'Get available date ranges for a specific supplier'
api: 'GET /api/v1/suppliers/{supplier_name}/date-ranges'
---

## Overview

This endpoint returns the available date ranges for a particular supplier. Use this to determine what time periods have data available before making data requests.

<Note>
  **Authorization Required** - You must include a valid Bearer token in the Authorization header. Contact [sales@rulebook.company](mailto:sales@rulebook.company) to obtain API access.
</Note>

## Path Parameters

<ParamField path="supplier_name" type="string" required>
  The name or code of the supplier (e.g., NYSE, NASDAQ, CBOE)
</ParamField>

## Request

```bash
curl -X GET https://api.rulebook.company/v1/suppliers/NYSE/date-ranges \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

## Response

<ResponseField name="supplier" type="string">
  Name of the supplier
</ResponseField>

<ResponseField name="date_ranges" type="array">
  Array of available date range objects

  <Expandable title="Date Range Object">
    <ResponseField name="start_date" type="string">
      Start date of available data (ISO 8601 format)
    </ResponseField>

    <ResponseField name="end_date" type="string">
      End date of available data (ISO 8601 format)
    </ResponseField>

    <ResponseField name="data_types" type="array">
      Types of data available in this range (e.g., fees, rebates, market data)
    </ResponseField>
  </Expandable>
</ResponseField>
