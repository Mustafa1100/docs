## Overview

This endpoint returns a comprehensive list of all exchanges available in the Rulebook Company database.

<Note>
  **Authorization Required** - You must include a valid Bearer token in the Authorization header. Contact [sales@rulebook.company](mailto:sales@rulebook.company) to obtain API access.
</Note>

## Request

```bash
curl -X GET https://api.rulebook.company/v1/exchanges \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

## Response

<ResponseField name="exchanges" type="array">
  Array of exchange objects

  <Expandable title="Exchange Object">
    <ResponseField name="id" type="string">
      Unique identifier for the exchange
    </ResponseField>

    <ResponseField name="name" type="string">
      Full name of the exchange
    </ResponseField>

    <ResponseField name="code" type="string">
      Exchange code (e.g., NYSE, NASDAQ)
    </ResponseField>

    <ResponseField name="country" type="string">
      Country where the exchange is located
    </ResponseField>

    <ResponseField name="status" type="string">
      Status of the exchange (active, inactive)
    </ResponseField>
  </Expandable>
</ResponseField>
