## Welcome to Rulebook Company API

The Rulebook Company API provides programmatic access to comprehensive exchange fee, rebate, and regulatory data across multiple trading venues. Our API enables you to retrieve historical and current fee schedules, analyze trading costs, and integrate exchange fee data into your applications.

<Note>
  **Get API Access** - To start using the Rulebook Company API, you need to contact our sales team to obtain your authorization token. Reach out at [sales@rulebook.company](mailto:sales@rulebook.company) or visit our [website](https://rulebookcompany.com/contact).
</Note>

## Base URL

All API requests should be made to:

```
https://api.rulebook.company/v1
```

## Authentication

All API endpoints require authentication using Bearer tokens. You must include your API key in the Authorization header of every request.

```bash
curl https://api.rulebook.company/v1/exchanges \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

Learn more about authentication in the [Authentication](/api-reference/authentication) section.

## Data Coverage

Our API provides access to:

- **Fee Types**: Options and Equity trading fees
- **Fee Categories**:
  - Fee And Rebates
  - Legal Regulatory Fees
  - Market Data Fees
  - Participant Fees
  - Port Fees And Other Services
- **Actions**: Make, Take, Open, Routed & Other
- **Participants**: Market Makers, Broker Dealers, Customers, Firms, and more
