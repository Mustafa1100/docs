## Sample API Calls

### Get Exchange Fee Schedule

Retrieve current fee schedules for a specific exchange:

<CodeGroup>

```bash cURL
curl https://api.rulebook.company/v1/exchanges/NASDAQ/fees \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

url = "https://api.rulebook.company/v1/exchanges/NASDAQ/fees"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```javascript JavaScript
const fetch = require('node-fetch');

const url = 'https://api.rulebook.company/v1/exchanges/NASDAQ/fees';
const headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
};

fetch(url, { headers })
    .then(response => response.json())
    .then(data => console.log(data));
```

</CodeGroup>

**Example Response:**
```json
{
  "exchange": "NASDAQ",
  "effective_date": "2024-01-01",
  "fee_schedule": {
    "maker_fee": -0.0002,
    "taker_fee": 0.0003,
    "tier_thresholds": {
      "tier_1": {
        "volume": 1000000,
        "maker_rebate": -0.00025
      },
      "tier_2": {
        "volume": 5000000,
        "maker_rebate": -0.0003
      }
    }
  },
  "last_updated": "2024-01-01T00:00:00Z"
}
```

### Historical Data Query

Query historical fee schedules:

<CodeGroup>

```bash cURL
curl "https://api.rulebook.company/v1/exchanges/NASDAQ/fees/historical?start_date=2023-01-01&end_date=2023-12-31" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```python Python
import requests

url = "https://api.rulebook.company/v1/exchanges/NASDAQ/fees/historical"
params = {
    "start_date": "2023-01-01",
    "end_date": "2023-12-31"
}
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

```javascript JavaScript
const params = new URLSearchParams({
    start_date: '2023-01-01',
    end_date: '2023-12-31'
});

const url = `https://api.rulebook.company/v1/exchanges/NASDAQ/fees/historical?${params}`;
const headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
};

fetch(url, { headers })
    .then(response => response.json())
    .then(data => console.log(data));
```

</CodeGroup>

## Rate Limits

When testing, be aware of API rate limits:

- **Standard tier**: 100 requests/minute
- **Professional tier**: 500 requests/minute
- **Enterprise tier**: Custom limits

<Tip>
Use the `X-RateLimit-Remaining` response header to track your remaining quota.
</Tip>
